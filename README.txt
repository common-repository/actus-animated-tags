== Description ==

Algorithmically Animated Tags.<br><br>
Random Post Tags or other taxonomies in different size, position and opacity, animated horizontally in various speed, over a background image or color.
<br>

* Animation of random Posts tags or other taxonomies against a background image or color
* Random position, size, opacity and speed for every term
* Adjust the height of the element.
* Adjust the density of the terms.
* Adjust the terms color.
* Six background images to choose from or upload your own image
* Set a color as a background
* Create and use different styles
* Use widget or shortcodes to embed the element anywhere in your pages.
* AJAX technology
* Plugin widget
* Plugin shortcode



== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the <b>Appearance > ACTUS Animated Tags</b> screen to configure the plugin
4. Use the shortcode <b>[actus_animated_tags]</b> to embed the animated tag cloud anywhere in your pages.
